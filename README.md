# Tenpista FullStack Challenge

Aplicación fullstack desarrollada como parte de un challenge técnico para el rol de TL FullStack.

## Stack Tecnológico

### Backend

- Java 21
- Spring Boot 3
- PostgreSQL 16

### Frontend

- React 18
- Vite
- Axios

### Infraestructura

- Docker
- Docker Compose

## Estructura del Proyecto

```text
tenpista-fullstack-challenge/
├── backend/
├── frontend/
├── docker-compose.yml
├── .env
└── README.md
```
